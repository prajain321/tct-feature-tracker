# **tct-feature-tracker**

## **Overview**

`tct-feature-tracker` is a Python-based application designed to track and analyze feature usage, including click analytics and UI enhancements. It provides tools for monitoring user interactions and updating data through scheduled cron jobs.

***

## **Features**

*   Click analytics tracking
*   UI enhancements for better user experience
*   Cron job integration for automated updates
*   Modular Python scripts for easy maintenance

***

## **Tech Stack**

*   **Language:** Python
*   **Dependencies:** Listed in `requirements.txt`

***

## **Installation**

```bash
# Clone the repository
git clone https://github.com/prajain321/tct-feature-tracker.git

# Navigate to the project directory
cd tct-feature-tracker

# Install dependencies
pip install -r requirements.txt
```

***

## **Usage**

Run the main application:

```bash
streamlit run app.py
```
